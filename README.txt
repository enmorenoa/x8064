     REQUISITOS PARA USAR LA HERRAMIENTA: Cualquier distribución de Ubuntu 20.04 LTS y utilizar la versión 12.1 de GDB.	
     Para poder ejecutar el proyecto correctamente, es necesario seguir los siguientes pasos:
     1º El usuario se tiene que situar en la carpeta raíz del proyecto, y abrir una terminal en esa ruta.
     2º En la terminal se tiene que ejecutar el archivo "install_dependencies.sh" con permisos de super usuario.
     3º Una vez instaladas las dependencias, para ejecutar el proyecto, la terminal se ubicará en la ruta raíz del
     proyecto, y se ejecutará "python3 x8064.py".
